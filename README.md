# monte-carlo-lab
Monte Carlo simulations for probability, stochastic processes, and decision modeling using Python and NumPy.
---
## 🎯 Goal

This project is a personal learning lab to build intuition for:

- Probability theory
- Random processes
- Expected value and variance
- Stochastic simulation
- Risk and uncertainty modeling

The focus is on understanding mathematical concepts through implementation.

---

## 🧠 Topics Covered

- Law of Large Numbers
- Monte Carlo estimation
- Random walks
- Expected value simulation
- Gambling and risk models
- Basic portfolio risk modeling

---

## 🚀 Why this project?

Monte Carlo methods are widely used in:

- Quantitative finance (risk modeling, pricing)
- Machine learning (uncertainty estimation)
- AI safety (robustness, stochastic modeling)
- Physics and simulation systems

This project is a step toward understanding these systems from first principles.
